📋 PM Design Notes

  • One-click issue categories (FR6.6) remove friction — customer doesn't need to write an essay to get help.
  • Photo upload is optional (FR6.7) — making it required would drop completion rates and add friction at the worst possible moment.
  • Submit button is disabled until an issue type is selected — reduces meaningless submissions.
  • Success state shows both reprint and cash refund simultaneously (FR6.3–6.5). "No vouchers" is explicit.
  • "Keep the original" removes the anxiety of having to return the product — no return label, no post office trip.
  • Issue data feeds back into AI training (FR3.9) and analytics (FR10.1–10.2) — shown here to close the loop with the customer.
